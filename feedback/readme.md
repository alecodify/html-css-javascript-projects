# Feedback

This project designed to collect and display user feedback through an intuitive form. It allows users to select the option to give the feedback and also give response on the given feedback.

## Installation

1. **Clone the repository:**
```bash
  git clone https://github.com/alecodify/html-css-javascript-projects.git
```
